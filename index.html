Recent Projects

Project 1

eSAS: Web-based Student Organization Management System

A streamlined platform for managing student organizations, enhancing club membership processes, and improving communication between students and moderators. The system allows administrators, moderators, and students to manage club memberships, view upcoming activities, and streamline communication. It simplifies organizational workflows, making it easier to track student participation and generate reports on club performance.

Project 2

Online Event Registration System

A web application allowing students to register and evaluate school events digitally, reducing manual work and improving data accuracy. Students can sign up for events, provide feedback, and access event information in real-time. The system reduces manual paperwork, provides organizers with detailed participant data, and offers insights through evaluation reports.

Project 3

Club Membership Tracker

A responsive system to monitor and manage club memberships, featuring real-time updates and member statistics for better organization. The system includes features like member profiles, attendance tracking, and real-time status updates. It empowers moderators to oversee membership details efficiently while offering students a transparent and accessible platform to engage with their organizations.
